## 가격 계산 공식

### Manual Shades
```
Base Price: $88.00
Price = Base Price + (Width in inches × Height in inches × $0.15)

Example:
48" W × 72" H = 3,456 sq in
Price = $88 + (3,456 × $0.15 / 144) = $88 + $36 = $124
```

### Motorized Shades (Standard)
```
Base Price: $168.00
Price = Base Price + (Width × Height × $0.20) + Motor Options + Remote Options + Accessories

Example:
48" W × 72" H with 5-Channel Remote
Price = $168 + (3,456 × $0.20 / 144) + $0 + $15 = $168 + $48 + $15 = $231
```

### Motorized Shades (Jacquard)
```
Base Price: $168.00
Price = Base Price + (Width × Height × $0.22) + Motor Options + Remote Options + Accessories

Example:
48" W × 72" H with Fabric-Wrapped Valance

```

## 설치 가이드

### Inside Mount
- Measure the exact width and height of the window opening
- Deduct 1/4" from width for proper clearance
- Recommended for windows with sufficient depth (2" minimum)

### Outside Mount
- Measure the area you want to cover
- Add 2-3" on each side for light blocking
- Add 2-3" to the top for headrail coverage

## 추천 사용처

### Manual Shades
- 예산 친화적인 솔루션
- 작은 창문
- 자주 조작하지 않는 공간
- 임대 주택

### Motorized Shades
- 높은 창문
- 여러 개의 창문 동시 제어
- 스마트 홈 통합
- 편의성 중시

### Motorized Jacquard
- 고급 인테리어
- 거실, 마스터 베드룸
- 디자인 중시
- 내구성 중시

## 유지보수

- 정기적인 먼지 제거 (부드러운 천 사용)
- 물세탁 불가
- 직사광선 장기 노출 주의
- 모터: 6개월마다 배터리 충전 권장
